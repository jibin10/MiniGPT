# Code of Conduct

MiniGPT is a learning project. The goal is to make the GPT training pipeline easier to understand for students, instructors, and self-learners.

Everyone who participates in this project is expected to be respectful and helpful.

## Expected behavior

Please:

- be kind and professional
- use welcoming language
- respect different levels of experience
- give feedback in a helpful way
- focus on the work, not the person
- be patient with learners and new contributors
- accept constructive feedback gracefully

It is okay to ask basic questions. It is also okay to make mistakes. This project is for learning.

## Unacceptable behavior

The following behavior is not acceptable:

- harassment or personal attacks
- insulting or disrespectful comments
- discrimination of any kind
- sexual or inappropriate language
- sharing private information without permission
- repeated unwanted contact
- intentionally disrupting discussions, issues, or pull requests

## Project discussions

Please keep discussions focused on the project.

Good topics include:

- improving the notebook
- fixing bugs
- making explanations clearer
- improving Colab reproducibility
- improving documentation for JOSE submission
- helping learners understand the model

If there is disagreement, discuss the idea calmly and with evidence.

## Reporting problems

If you see behavior that violates this Code of Conduct, please report it to the project maintainer.

You can contact:

Jibin Joseph  
jibinjoseph@utexas.edu

Please do not open a public GitHub issue for private or sensitive conduct reports.

## Enforcement

The maintainer may take action if this Code of Conduct is violated.

Possible actions include:

- asking the person to stop the behavior
- removing inappropriate comments
- closing issues or pull requests
- blocking a user from the repository
- taking any other reasonable action needed to keep the project respectful

Reports will be handled as privately as possible.
